Welcome to the Cafe & Wifi API

This API provides endpoints to manage cafe information including location, wifi availability, and pricing.

Available Endpoints

  • GET /random - Get a random cafe
  • GET /all - Get all cafes
  • GET /search - Search cafes by location
  • POST /add - Add a new cafe
  • PATCH /update-price/<id> - Update cafe price
  • DELETE /report-closed/<id> - Report cafe as closed
